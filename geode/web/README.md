# Geode Web

This folder is reset to a minimal Next.js scaffold.

The previous Geode UI has been removed. The remaining frontend keeps only the app shell required
for a clean rebuild, the TypeScript and Tailwind configuration, and `src/lib/paths.ts`.

## Start

```bash
pnpm install
pnpm run dev
```

The app starts at `http://localhost:3000`.

## Verify

```bash
pnpm run build
pnpm run lint
```
