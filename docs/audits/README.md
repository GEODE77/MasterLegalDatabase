# Geode Audit Notes

This directory stores human-readable audit reports and language corrections.
Reports in this folder are dated historical evidence. Older reports may mention
former frontend, route, page, forum, or visual-design work; those references are
historical only and do not describe current Geode architecture.

Use this folder for summaries that explain audit findings, completion-report wording
changes, route-by-route review notes, and comparison notes against reference sites.
Every material claim should point back to source files or control-plane evidence.
